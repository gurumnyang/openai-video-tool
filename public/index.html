OpenAI Video Generator

프롬프트(및 선택 이미지 레퍼런스)로 비디오 생성 작업을 만들고, 진행 상황을 확인한 뒤 MP4를 다운로드합니다.

생성 요청

이 페이지는 기본적으로 같은 도메인의 /api 프록시(Cloudflare Pages Functions)를 통해 OpenAI API를 호출합니다.

이미지 한 장을 첨부하면 생성 결과를 가이드할 수 있습니다.
API Key는 이 브라우저의 로컬 저장소에만 저장됩니다. 공용 PC/공유 계정에서는 사용을 피하세요.
완료될 때까지 지정한 간격으로 상태를 조회합니다.
